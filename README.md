# 지뢰찾기 (Minesweeper)

빌드 도구나 프레임워크 없이 순수 HTML/CSS/JavaScript로 만든 브라우저용 지뢰찾기 게임입니다.
파일을 그대로 정적 서버에 올리기만 하면 바로 플레이할 수 있습니다.

## 데모 화면

| 라이트 모드 | 다크 모드 |
| --- | --- |
| 보드, 남은 지뢰 수, 타이머, 최고 기록이 상단에 표시됩니다 | 우측 상단 🌙 버튼으로 다크모드 전환 가능 |

## 주요 기능

- **난이도 선택**
  - 초급: 9x9, 지뢰 10개
  - 중급: 16x16, 지뢰 40개
  - 고급: 16x30, 지뢰 99개
- **첫 클릭 안전 보장**: 처음 클릭한 칸과 그 주변에는 지뢰가 배치되지 않습니다.
- **좌클릭으로 칸 열기**, 인접 지뢰가 0인 칸은 주변 칸까지 자동으로 열립니다(flood fill).
- **우클릭으로 깃발 표시/해제**, 남은 지뢰 수 카운터에 반영됩니다.
- **승리 / 패배 판정**: 지뢰를 밟으면 전체 지뢰가 공개되고, 안전한 칸을 모두 열면 승리 모달이 뜹니다.
- **타이머**: 첫 클릭 시 시작하고 게임이 끝나면 정지합니다.
- **최고 기록 저장**: 난이도별 최단 클리어 시간을 `localStorage`에 저장합니다.
- **다크모드**: 설정이 `localStorage`에 저장되어 다음 방문 시에도 유지됩니다.
- **반응형 레이아웃**: 화면 크기에 맞춰 보드 셀 크기가 자동으로 조정됩니다.

## 게임 방법

1. 상단 드롭다운에서 난이도를 선택합니다.
2. 원하는 칸을 좌클릭하면 게임이 시작되고 타이머가 작동합니다.
3. 지뢰로 의심되는 칸은 우클릭으로 깃발을 꽂아 표시합니다.
4. 지뢰가 아닌 모든 칸을 열면 승리, 지뢰를 열면 패배입니다.
5. 상단 중앙의 스마일 버튼(🙂)을 누르면 같은 난이도로 새 게임을 시작합니다.

## 실행 방법

빌드 과정이 필요 없습니다. 브라우저의 ES Module 정책상 `index.html`을 파일로 직접 여는 대신
간단한 정적 서버로 실행하는 것을 권장합니다.

```bash
# Node.js가 있는 경우
npx serve .

# Python이 있는 경우
python -m http.server 8080
```

서버 실행 후 브라우저에서 `http://localhost:8080` (또는 안내된 주소)로 접속하세요.

## 기술 스택

- HTML5 / CSS3 / Vanilla JavaScript (ES Modules)
- 외부 라이브러리, 프레임워크, 번들러 없음
- 데이터 저장: 브라우저 `localStorage`

## 브라우저 호환성

ES Modules(`<script type="module">`)를 사용하므로 최신 Chrome, Edge, Firefox, Safari 등
모던 브라우저에서 정상 동작합니다.
